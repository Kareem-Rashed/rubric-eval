Why Rubric

Built for developers.
Not for any AI company.

After Promptfoo joined OpenAI, the community needed a truly independent evaluation framework. Rubric is that answer — open source forever, with no cloud lock-in.

🤖

First-Class Agent Evals

Evaluate multi-step reasoning, tool call sequences, trace quality, latency budgets, and cost — not just final outputs.

🔌

Works With Any LLM

Pass any callable as your judge function. Native support for OpenAI, Anthropic, and Ollama — or wire up your own.

📦

Zero Required Dependencies

Core string metrics work out of the box. Install only what you need via extras: [semantic], [openai], [anthropic].

🧪

Native pytest Integration

Drop-in rubric_eval fixture. Your LLM evals live alongside your unit tests — in the same pytest run.

📊

Beautiful HTML Reports

Self-contained, interactive HTML dashboards. No server required. Filter by pass/fail, drill into metric breakdowns.

🛡️

Neutral & Independent

MIT licensed. Not owned by OpenAI, Anthropic, or any model provider. Evaluate any model with the same unbiased framework.

Quick Start

Up and running
in 5 minutes.

Install, write test cases, run metrics. That's it.

basic_eval.py 
import rubriceval as rubric

# Your LLM under test
def my_llm(prompt):
    return "The capital of France is Paris."

results = rubric.evaluate(
    test_cases=[
        rubric.TestCase(
            input="What is the capital of France?",
            actual_output=my_llm("..."),
            expected_output="Paris",
        )
    ],
    metrics=[
        rubric.Contains("Paris"),
        rubric.NotContains("I don't know"),
        rubric.LLMJudge(criteria="Is the answer correct?"),
    ],
    output_html="report.html",
)

agent_eval.py 
import rubriceval as rubric

results = rubric.evaluate(
    test_cases=[
        rubric.AgentTestCase(
            name="flight booking",
            input="Book a flight to Tokyo",
            actual_output=agent_output,
            expected_tools=[
                "search_flights",
                "check_availability",
                "book_flight",
            ],
            forbidden_tools=["send_email"],
            tool_calls=agent.tool_calls,
            trace=agent.trace,
            latency_ms=1240,
            cost_usd=0.004,
        )
    ],
    metrics=[
        rubric.ToolCallAccuracy(check_order=True),
        rubric.TraceQuality(penalize_loops=True),
        rubric.SafetyCompliance(),
        rubric.LatencyMetric(max_ms=5000),
        rubric.CostMetric(max_cost_usd=0.01),
    ],
    output_html="agent_report.html",
)

results.print_summary()
